Question-1: What are the different ways to select an element in the DOM?

Here are the main ways to select an element in the DOM:
1.getElementById() - Selects an element by its ID.
2.getElementsByClassName() - Selects elements by their class name.
3.getElementsByTagName() - Selects elements by their tag name.
4.querySelector() - Selects the first element that matches a CSS selector.
5.querySelectorAll() - Selects all elements that match a CSS selector.
6.parentNode.querySelector() - Selects an element within a specific parent.
7.childNodes - Accesses child elements directly.

Question-2: What is the difference between innerHTML, innerText, and textContent?

Key difference between innerHTML, innerText, and textContent innerHTML
1. sets the HTML content inside an element, including tags.
2. innerText:sets the visible text inside an element, excluding HTML tags and styles.
3. textContent:sets the text content inside an element, including hidden text.

Question-3: What is event delegation in the DOM?

ইভেন্ট ডেলিগেশন হল একটি টেকনিক যেখানে আপনি একটি প্যারেন্ট এলিমেন্টে ইভেন্ট লিসনার অ্যাটাচ করেন, এবং সেই প্যারেন্ট এলিমেন্টের মাধ্যমে তার সমস্ত চাইল্ড এলিমেন্টে ইভেন্ট প্রোপাগেট হয়। এটি DOM-এ ইভেন্ট হ্যান্ডলিংয়ের কার্যকারিতা বাড়ায় এবং মেমরি ব্যবহারে সুবিধা দেয়।

ইভেন্ট বাবলিং হল একটি প্রক্রিয়া যেখানে ইভেন্টটি টার্গেট এলিমেন্ট থেকে শুরু হয়ে প্যারেন্ট এলিমেন্টগুলোর মাধ্যমে উপরে উঠে চলে, এবং প্রতিটি প্যারেন্টে একই ইভেন্ট ট্রিগার হয়।

Question-5: How do you create, add, and remove elements using JavaScript?

Create an element:
let newElement = document.createElement("div");
newElement.textContent = "Hello!";

Add (append) the element to the page:
document.body.appendChild(newElement);

Remove an element:
newElement.remove();
